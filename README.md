# Undergraduate Research under Dr. Joel Liebman

This repository contains code developed during my undergraduate research at the University of Maryland, Baltimore County (UMBC) under Dr. Joel Liebman.

## Overview
The work focuses on computational modeling of molecular systems, including matrix-based methods and eigenvalue/eigenvector analysis to study molecular structure and symmetry.

## Methods and Tools
- MATLAB
- Numerical modeling
- Linear algebra techniques

## Context
This research contributed to a peer-reviewed publication in a Springer Nature volume on molecular structure and hybridization theory.
